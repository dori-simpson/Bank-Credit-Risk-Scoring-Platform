# Bank Credit Risk Scoring Platform ⬡

An interactive machine learning decision engine and Streamlit dashboard designed to quantify retail loan default risk.

## 🚀 Key Features
* **Probability of Default (PD) Scoring Engine:** Utilizes a standard-scaled Logistic Regression classification engine.
* **Exploratory Analytics Dashboard:** Interactive Streamlit-powered interface rendering loan distributions and risk tiers.
* **Portfolio-Level Risk Analysis:** Aggregates exposures to estimate portfolio Expected Loss ($EL = PD \times EAD$).
